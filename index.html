Game audio / Integration / Sound design / Music

Hi !
I am currently working at Game Audio Factory (Angoulême, France) mainly as a game sound designer.

My job includes :

-Audio asset creation (SFX, Ambience, Interface, Foley) using DAWs, soundbanks, recording and plugins

-Audio integration in the game engine using middlewares such as Wwise

-Actor recording

-Voice cleaning and batch processing

-Vocal reduction of music tracks for karaoke games

-Linear audio editing and mixing (mainly game trailers, shorts and corporate video)

Recent Work

Drive Another Day

// Sound design, Integration (FMOD Studio), Nodal programming, Music

A cheesy 80s humoristic car runner on tablet with gadgets and an unsympathetic AI. As I wanted the audio to feel as energetic as possible, the game features half cartoonish/hollywoodian SFX, diverse radio stations and a dynamic real-time mixing hierarchy.

Reach

// Sound design, Music, Audio Integration (Wwise)

Reach is a student experimentation of incarnation and flight inside a virtual reality. The player sits in a custom-made chair. The installation reproduces all the player’s movements thanks to Arduino components, a Razer Hydra and an Oculus Rift. The game features an evocative sound aesthetic and tempo-based sound design which is heavily linked to the music.

Soon

// Sound design, Audio Integration (Fabric)

Manipulate time to solve puzzles. This ambient game places you in a glitchy synthetically reproduced natural ecosystem. The time variable allowed me to conceive a dynamic ambience/SFX integration system.

Teyolia

// Sound design, Music, Audio Integration (XNA)

A XNA-based two button, retro arcade/reflex game which involves knocking parts of totems to earn the highest amount of points possible. It features a hybrid sound aestetic between modernism and retrogaming and diverse layered music evolving with the combo counter. This game was initially made for GBJam 2013, and we are currenty remaking it with a more modern aesthetic for android devices.

ECGr

// Sound design, Audio Integration (PD), Nodal programming

ECGr is a self-made pure data rhythm/memory game featuring real-time sound synthesis. In this game, you have to save a patient from a heart attack by pressing the correct button at the right time.

WYSIWYC

/ Sound design, Music

WYSIWYC (What you see is what you code) is an educational video game focused on teaching in a fun way the fundamentals of programming. The game is aesthetically inspired by retro electronics and 90's documentaries. We wanted to communicate a feeling of nostalgia and "cathodic utopia". I composed an evolutive music which follows your progression and designed the sound assets.
